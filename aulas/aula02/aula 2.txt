npm -> gerenciador de pacotes padrao do nodejs, a função é instalar, atualizar e gerenciar as dependencias de um projeto

existe um site chamado npm que tem varios pacotes do que da pra instalar la

instalamos o bootstrap usando: npm i bootstrap

instalamos o input is que é a validação de inputs

pra baixar as configuracoes e pacotes de um projeto colocar: npm i OU npm install

pra iniciar o projeto eu coloco "npm run dev" pq o dev criado no package.json é o comando que o programa precisa pra saber qual arquivo vai abrir que no caso é o index.js

o cmd1 roda o arquivo contador e tudo oq tem dentro do arquivo

resumindo:
- npm init //inicia o package.json
- npm install / npm i
- npm update //atualiza os pacotes no projeto
- npm run //roda o projeto
